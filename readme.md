# SEO Super Engine V6.5 - Word 文档自动化审计与修复系统

![Python Version](https://img.shields.io/badge/python-3.8%2B-blue)
![Dependencies](https://img.shields.io/badge/dependencies-python--docx%20%7C%20pandas%20%7C%20lxml-green)

企业级 Word 文档 SEO 自动化审查与修复引擎。旨在通过并行化处理、沙箱隔离以及精准的 DOM 操作，对大规模 SEO 文章进行自动化质量控制（QC）、清洗和最终交付物打包。

## 🚀 核心特性 (Features)

- **外链智能清洗**: 无损剥离 Word 中的标准超链接与隐藏的域代码 (Field Codes) 链接，精准保留纯文本与原生排版格式。
- **SEO 标题层级修复**: 引入动态相对偏移量 (Level Tracker) 算法，智能修正 `H1 -> H3 -> H3` 等不合规跳级，自动映射回 `H1 -> H2 -> H2`，且完美维持后续合法嵌套关系。
- **零破坏沙箱隔离**: (Zero-Destruction Sandbox) 所有修复与修改操作均在 `_Cleaned_Output` 镜像目录中进行，绝不污染或损坏原始文档源文件。
- **细粒度跳过规则**: 支持基于 `skip_rules.json` 实现文件级别的豁免控制（可针对特定关键词文件单独关闭 H1 检查或外链清理）。
- **一键提取与打包交付**: 修复完成后，自动将洁净的 `.docx` 与对应原图（智能过滤截屏等杂图）剥离并按原目录结构打包至桌面交付文件夹。
- **高并发架构**: 底层基于 `ThreadPoolExecutor` 实现了多线程 IO 并发，轻松应对包含数千个 Word 文档的庞大目录结构。

## ⚙️ 配置文件: skip_rules.json

您可以在需要扫描的根目录下放置 `skip_rules.json` 文件，实现细粒度规则控制。支持的 `skip_checks` 包含：
- `h1_check`: 跳过 H1 唯一性检查
- `links_clean`: 跳过外链清理
- `heading_fix`: 跳过标题层级跳级修复
- `image_fix`: 跳过图片后缀与降级修复

**示例**:
```json
{
  "rules": [
    {
      "keywords": ["draft", "reference"],
      "skip_checks": ["h1_check", "links_clean"]
    }
  ]
}
```

## 📖 简介
SEO Super Engine 是一款专为大规模内容运营打造的 Python 自动化脚本。它能够深度解析 Word (`.docx`) 文档的底层 XML 结构，实现 SEO 规范的毫秒级审计，并提供自动化修复功能。

**V6.4 核心突破：** 引入了“域代码级 (Field Codes) 外链粉碎”与“无痕格式融合”技术，彻底解决顽固外链与清理后排版突兀的问题。

## ✨ 核心特性
* **🔗 深度外链清洗：** 三轨并发（标准节点、隐藏域代码、纯文本），精准识别并剥离非本站外链。
* **💅 无痕排版融合：** 移除超链接后，自动卸载底层文本的专属颜色（如蓝色）和下划线，完美继承周边正文排版。
* **🛡️ 映射健康度熔断：** 强制校验 Excel 域名映射表。当文件夹匹配失败率过高时，触发预警阻断，防止静默失效。
* **📑 智能层级修复：** 自动检测 H1-H6 标题跳级现象，并==将误设为标题的图片标注==降级为正文。
* **🖼️ 资产完整性校验：** 自动比对文档内引用的图片与实际文件夹内的 `.webp` 资产，定位缺失文件。
