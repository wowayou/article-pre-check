# 🚀 SEO Super Engine V6.4 - 自动化审计与无痕修复引擎

## 📖 简介
SEO Super Engine 是一款专为大规模内容运营打造的 Python 自动化脚本。它能够深度解析 Word (`.docx`) 文档的底层 XML 结构，实现 SEO 规范的毫秒级审计，并提供自动化修复功能。

**V6.4 核心突破：** 引入了“域代码级 (Field Codes) 外链粉碎”与“无痕格式融合”技术，彻底解决顽固外链与清理后排版突兀的问题。

## ✨ 核心特性
* **🔗 深度外链清洗：** 三轨并发（标准节点、隐藏域代码、纯文本），精准识别并剥离非本站外链。
* **💅 无痕排版融合：** 移除超链接后，自动卸载底层文本的专属颜色（如蓝色）和下划线，完美继承周边正文排版。
* **🛡️ 映射健康度熔断：** 强制校验 Excel 域名映射表。当文件夹匹配失败率过高时，触发预警阻断，防止静默失效。
* **📑 智能层级修复：** 自动检测 H1-H6 标题跳级现象，并==将误设为标题的图片标注==降级为正文。
* **🖼️ 资产完整性校验：** 自动比对文档内引用的图片与实际文件夹内的 `.webp` 资产，定位缺失文件。

## ⚙️ 快速开始
1.  **环境准备：** 确保已安装 Python，并执行 `pip install python-docx pandas openpyxl lxml`。
2.  **配置映射表：** 准备 `域名映射.xlsx`，**第一列必须为本地电脑上的真实文件夹名称**（如：`佛山科耐`），第二列为目标域名（如：`konaz.cn`）。
3.  **运行脚本：** 运行 `pre-check-article_v6-4.py`，根据弹窗提示依次：加载映射表 -> 选择文章总目录 -> 确认外链清理策略 -> 执行初审与修复。
